Full Stack · Meal Ordering Web Application

PROJECT 02
Fresh Plate & Co. — Meal Ordering Web Application
  • Built a full-stack meal-ordering platform with a Node.js/Express backend and a hand-coded HTML/CSS/JS frontend, with no external UI framework.
  • Implemented JWT authentication with bcrypt password hashing and separate access levels for customers and administrators.
  • Built an admin dashboard on REST APIs to track daily revenue, manage incoming orders, and update order status in real time.
  • Added an automatic in-memory fallback store alongside MongoDB/Mongoose so the site keeps accepting orders through a database outage.
  • Proxied a third-party AI chat service through the backend to eliminate frontend CORS issues and keep API keys off the client.
  • Wrote a Jest/Supertest integration test suite covering health checks, order validation, user registration, and JWT-guarded admin routes, and set up a GitHub Actions CI pipeline that runs the suite on Node 18.x and 20.x for every push/PR.
  • Added request validation middleware (email/phone/price checks) and tiered rate-limiting via express-rate-limit (100 req/15min general, 15 req/15min on auth) to harden endpoints against abuse.
  • Stack: Node.js · Express.js · MongoDB · Mongoose · JWT · HTML/CSS/JS
